# 생각구독 콘텐츠 분석 에이전시 v2 — 사용 매뉴얼

유튜브 영상 스크립트를 OpenAI로 분석하여 유료 구독 전환률을 높이기 위한 인사이트를 축적하는 도구입니다.
**CLI 모드**와 **웹 모드** 두 가지 방식으로 실행할 수 있습니다.

## 1. 사전 요구사항

| 항목 | 요구 버전 |
|------|-----------|
| Python | 3.10 이상 |
| Anthropic API 키 | [console.anthropic.com](https://console.anthropic.com) 에서 발급 |
| 인터넷 연결 | 유튜브 자막 추출 및 Claude API 호출에 필요 |

> **자막이 없는 영상은 분석할 수 없습니다.** 유튜브 자막(수동/자동 생성 모두 지원)이 활성화된 영상만 사용 가능합니다.

## 2. 설치

```bash
# 프로젝트 디렉토리로 이동
cd saenggak-agency

# 가상환경 생성 (권장)
python -m venv venv

# 가상환경 활성화
# macOS / Linux
source venv/bin/activate
# Windows
venv\Scripts\activate

# 의존성 설치
pip install -r requirements.txt
```

설치되는 주요 패키지:

| 패키지 | 역할 |
|--------|------|
| `anthropic` | Claude AI API 호출 |
| `youtube-transcript-api` | 유튜브 자막 추출 |
| `yt-dlp` | 영상 메타데이터(제목, 태그 등) 추출 |
| `rich` | 터미널 UI (테이블, 진행바 등) |
| `pandas` | 데이터 집계 |
| `matplotlib` | 레이더 차트 생성 |
| `rapidfuzz` | 유사 패턴 병합 |
| `python-dotenv` | `.env` 환경변수 로드 |

## 3. 환경 설정

프로젝트 루트의 `.env.example`을 복사하여 `.env` 파일을 만듭니다.

```bash
cp .env.example .env
```

`.env` 파일을 열어 아래 값을 수정합니다.

```dotenv
# 필수: Anthropic API 키
ANTHROPIC_API_KEY=sk-ant-여기에_실제_키_입력

# 선택: DB 저장 경로 (기본값 사용 권장)
DB_PATH=database/saenggak.db

# 선택: 사용할 Claude 모델
CLAUDE_MODEL=claude-sonnet-4-6

# 선택: 응답 온도 (0 = 일관성 최대)
CLAUDE_TEMPERATURE=0

# 선택: 유사 패턴 병합 임계값 (0~100, 기본 80)
PATTERN_SIMILARITY_THRESHOLD=80
```

> `.env` 파일은 `.gitignore`에 포함되어 있어 Git에 커밋되지 않습니다. API 키가 외부에 노출되지 않도록 주의하세요.

## 4. 실행

### 웹 모드 (권장)

```bash
# venv 활성화 후 웹 서버 실행
source venv/bin/activate
uvicorn app:app --host 0.0.0.0 --port 8000 --reload

# 브라우저에서 접속
open http://localhost:8000
```

브라우저에서 7개 탭 (대시보드 / 신규 분석 / 영상 검색 / 패턴 / 트렌드 / 심화 인사이트 / 설정) 을 통해 모든 기능을 사용할 수 있습니다.

외부 공개 배포 시에는 `--reload` 플래그를 제거하고, 도메인이나 IP 주소를 `--host` 옵션에 지정하세요.

## 5. 메뉴 별 사용법

### [1] 신규 분석

유튜브 영상을 분석하고 결과를 DB에 저장합니다. 한 번에 최대 10개까지 입력할 수 있습니다.

**사용 흐름:**

```
📹 신규 분석
유튜브 링크와 구독전환률을 입력하세요. (최대 10개, 빈 줄 입력 시 종료)

[영상 1]
  유튜브 링크 (빈 줄 = 종료): https://www.youtube.com/watch?v=XXXXXXXXXXX
  구독전환률 (%): 3.5

[영상 2]
  유튜브 링크 (빈 줄 = 종료): https://youtu.be/YYYYYYYYYYY
  구독전환률 (%): 1.2

[영상 3]
  유튜브 링크 (빈 줄 = 종료):   ← 빈 줄 입력 시 분석 시작
```

**구독전환률 입력 방법:**
- YouTube Studio의 해당 영상 → "분석" → "도달범위" 탭에서 확인
- 해당 기간 구독자 유입 수 ÷ 조회수 × 100 (%)

**분석 결과:**
- 7개 차원 점수 (1~10점): hook / story / value / cta / emotion / topic / diff + overall
- `outputs/reports/` 에 마크다운 리포트 저장
- 5개 이상 영상 축적 시: 레이더 차트(`outputs/charts/`) 자동 생성

**이미 분석한 영상 재분석:**
- 동일 URL 입력 시 "재분석 하시겠습니까?" 메시지 출력
- `y` 입력 → 기존 패턴 삭제 후 재분석
- `n` 입력 → 전환률만 업데이트하고 기존 분석 유지

---

### [2] 대시보드

축적된 전체 데이터의 현황을 한눈에 확인합니다.

**표시 내용:**
- 총 분석 영상 수, 평균/중앙값/최고/최저 전환률, 표준편차
- 전환률 TOP 5 영상 / BOTTOM 5 영상 테이블
- 최근 분석 세션 이력 (최대 5건)

> 분석된 영상이 없으면 대시보드에 데이터가 표시되지 않습니다. 먼저 [1] 신규 분석을 진행하세요.

---

### [3] 트렌드 분석

시간에 따른 전환률 추이와 패턴 간 상관관계를 분석합니다.

**최소 데이터:** 8개 영상 이상 필요

**분석 내용:**
- 시간대별 전환률 추이
- 성공/실패 패턴 빈도 변화
- 점수 항목별 상관관계 (예: hook이 높을수록 전환률이 높은지)

---

### [4] 영상 검색

4가지 방법으로 과거에 분석한 영상을 검색합니다.

| 선택 | 검색 방법 | 예시 |
|------|-----------|------|
| `1` | 키워드 검색 | 제목/스크립트/태그에서 "독서" 검색 |
| `2` | 전환률 범위 | 전환률 2.0% ~ 5.0% 사이 영상 |
| `3` | 점수 기준 | hook ≥ 7점 AND cta ≥ 6점인 영상 |
| `4` | 전체 목록 | 모든 영상 보기 |

검색 결과 목록에서 번호를 입력하면 해당 영상의 **상세 분석 내용**을 확인할 수 있습니다.

---

### [5] 심화 인사이트

축적된 데이터를 종합하여 채널 전략 방향을 제시합니다.

**최소 데이터:** 10개 영상 이상 필요

**분석 내용:**
- 성공 패턴 / 실패 패턴 종합 정리
- 전환률을 높이는 핵심 요인 도출
- 앞으로의 콘텐츠 제작 방향 제언
- 결과는 `outputs/trends/` 에 마크다운 리포트로 저장

---

### [6] 데이터 내보내기

분석 데이터를 외부 파일로 저장합니다.

| 선택 | 형식 | 저장 위치 | 용도 |
|------|------|-----------|------|
| `1` | CSV | `data/export_YYYY-MM-DD.csv` | 엑셀/구글시트 활용 |
| `2` | JSON | `data/export_YYYY-MM-DD.json` | 개발/백업 |
| `3` | 마크다운 | `data/videos/` (영상별) + `data/index.md` | 아카이브/문서화 |

---

### [7] 설정

DB 데이터를 관리합니다.

| 선택 | 기능 | 주의사항 |
|------|------|----------|
| `1` | 전환률 수정 | 유튜브 링크 또는 video_id 입력 |
| `2` | 영상 삭제 | 연관 패턴까지 함께 삭제됨 |
| `3` | 패턴 초기화 | **전체 패턴 삭제** — 복구 불가 |
| `4` | DB 백업 | `data/` 폴더에 `.db` 파일 복사 |
| `5` | DB 통계 | 테이블별 레코드 수 확인 |

> **패턴 초기화(3번)**는 되돌릴 수 없습니다. 먼저 DB 백업(4번)을 진행하세요.

## 6. 출력 파일 구조

```
saenggak-agency/
├── database/
│   └── saenggak.db          ← SQLite DB (영상/분석/패턴/세션 저장)
├── outputs/
│   ├── reports/             ← 신규 분석 리포트 (마크다운)
│   │   └── report_YYYY-MM-DD_HH-MM-SS.md
│   ├── trends/              ← 심화 인사이트 리포트
│   │   └── insight_YYYY-MM-DD.md
│   └── charts/              ← 레이더 차트 (PNG)
│       └── radar_YYYY-MM-DD.png
└── data/
    ├── export_YYYY-MM-DD.csv   ← CSV 내보내기
    ├── export_YYYY-MM-DD.json  ← JSON 내보내기
    ├── index.md                ← 마크다운 아카이브 인덱스
    └── videos/
        └── VIDEO_ID.md         ← 영상별 마크다운 분석
```

## 7. 운영 팁

**데이터 축적 로드맵**

| 영상 수 | 활성화되는 기능 |
|---------|----------------|
| 1개~ | 신규 분석, 대시보드, 검색, 내보내기 |
| 5개~ | 역사 컨텍스트 반영 분석, 레이더 차트 비교 |
| 8개~ | 트렌드 분석 |
| 10개~ | 심화 인사이트 |

**정기 운영 루틴 예시**
1. 영상 업로드 후 일주일 뒤 구독전환률 확정 → [1] 신규 분석 실행
2. 월 1회 [5] 심화 인사이트로 채널 전략 점검
3. 분기 1회 [6] 데이터 내보내기 → CSV로 외부 아카이브

**DB 백업**
- 운영 중 DB 손상에 대비해 [7] → [4] 백업을 주기적으로 실행하세요.
- `database/saenggak.db` 파일을 직접 복사해도 됩니다.

## 8. 자주 묻는 질문

**Q. "자막 없음" 오류가 계속 납니다.**
→ 해당 영상에 유튜브 자막(수동 또는 자동 생성)이 없는 경우입니다. 영상 자체에서 자막 설정을 먼저 확인하세요.

**Q. 비공개 영상을 분석할 수 있나요?**
→ 불가능합니다. 공개 또는 일부 공개 영상만 지원합니다.

**Q. API 비용은 얼마나 드나요?**
→ 영상 1개 분석에 약 4,000~6,000 토큰이 소비됩니다. Anthropic 요금제에 따라 다르며, [console.anthropic.com](https://console.anthropic.com) 에서 실시간 사용량을 확인할 수 있습니다.

**Q. 분석 모델을 바꾸고 싶습니다.**
→ `.env`의 `CLAUDE_MODEL` 값을 변경하세요. (예: `claude-opus-4-6`, `claude-haiku-4-5-20251001`)

**Q. 패턴 유사도 기준을 조정하고 싶습니다.**
→ `.env`의 `PATTERN_SIMILARITY_THRESHOLD` 값을 조정하세요. 숫자가 높을수록 더 엄격하게 구분합니다 (기본값: 80).

**Q. DB를 초기화하고 싶습니다.**
→ `database/saenggak.db` 파일을 직접 삭제하면 다음 실행 시 빈 DB가 자동 생성됩니다.
